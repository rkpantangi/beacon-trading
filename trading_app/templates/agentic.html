Agentic Trading

Connect your favorite agentic development tool directly to your Beacon Trading paper trading account.

Model Context Protocol (MCP)

Beacon Trading supports the Model Context Protocol (MCP), an open standard that allows AI models and coding assistants to securely access local or remote tools.

By connecting the Beacon Trading MCP server, your agent can query market data, manage your portfolio, and place trades against your account.

  • 📊 Query Market Data: Get real-time stock quotes.
  • 💼 Manage Portfolio: View your account balances and current holdings.
  • 📈 Place Trades: Execute market buy and sell orders.
  • 📝 Order Book: List, review, and cancel active limit or market orders.

Integrate with Claude (Code / Desktop)

To add the Beacon Trading tools to Claude, run this command in your terminal:

Terminal Command 
claude mcp add --scope user --transport http beacon-trading https://agent-beacon-trading-paper.fly.dev/mcp

Integrate with Google Antigravity

To add the tools to Google Antigravity, add the server configuration to your global config file at ~/.gemini/config/mcp_config.json (or locally in your workspace under .agents/mcp_config.json):

mcp_config.json Configuration 
{
  "mcpServers": {
    "beacon-trading": {
      "serverUrl": "https://agent-beacon-trading-paper.fly.dev/mcp"
    }
  }
}

Exposed Tools Reference

Once connected, your agent will have access to the following functions grouped by category:

🔍 Market Data & Search

get_equity_quotes

Retrieve real-time price info, daily change, and volume details for stock symbols.

search

Search for stock instruments and metadata by ticker symbol or company name.

💼 Portfolio & Positions

get_portfolio

Get account balance, total equity, buying power, and daily/unrealized profit & loss summary.

get_equity_positions

List all current stock holdings, average entry price, total value, and unrealized gains/losses.

📈 Order Management

get_equity_orders

Retrieve the log of all orders (open, filled, or canceled) with type, side, quantity, and status details.

review_equity_order

Submit order details for validation. Generates a preview and a 5-minute confirmation token.

place_equity_order

Execute and place an order on the exchange using a valid order confirmation token.

cancel_equity_order

Cancel an active, pending limit or stop order before it executes.

⭐ Watchlist Management

get_watchlist

List the account's saved watchlist symbols, each combined with its current market quote.

add_to_watchlist

Save a stock symbol to your account's watchlist.

remove_from_watchlist

Remove a stock symbol from your account's watchlist.

💵 Funding & Transfers

deposit

Deposit simulated cash funds into a trading account.

withdraw

Withdraw simulated cash funds from a trading account.
